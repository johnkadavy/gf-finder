# 🎨 Design System

## Vibe
- Clean, modern, trustworthy
- Health-focused (not sterile SaaS)

## UI Framework
- shadcn/ui (primary component system)
- Tailwind CSS

## Colors
- Primary: Green (safety)
- Secondary: Yellow (caution)
- Danger: Red (risk)
- Neutral: Tailwind gray scale

## Typography
- Font: Inter
- Emphasis on readability + hierarchy

## Core Components

### Search Bar
- Large, centered
- Rounded-xl
- Subtle shadow
- Icon inside

### Restaurant Card
- Card-based layout
- Includes:
  - Name
  - Location
  - Safety score (prominent)
  - Short summary

### Safety Score (Key Differentiator)
- Visual first (not text-first)
- Color-coded:
  - Green = Safe
  - Yellow = Moderate risk
  - Red = High risk
- Consider:
  - Badge
  - Progress bar
  - Letter grade

## Design Principles
- Prioritize clarity over density
- Use whitespace generously
- Avoid clutter
- Default to existing components (don’t reinvent UI)

## 🎯 Goal
Make gluten-free safety instantly understandable at a glance.
